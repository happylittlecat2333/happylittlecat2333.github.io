Emacs Cheat Sheet

Basic

  • C-x C-f 打开/新建文件
  • C-x C-s 保存当前缓冲区
  • C-x C-w 当前缓冲区另存为
  • C-x C-v 关闭当前Buffer并打开新文件
  • C-x i 光标处插入文件
  • C-x b 切换Buffer
  • C-x C-b 显示Buffer列表
  • C-x k 关闭当前Buffer
  • C-x C-c 关闭Emacs
  • C-c C-z 终止shell中的进程

Cursor

  • C-f 前进一个字符
  • C-b 后退一个字符
  • C-p 上一行
  • C-n 下一行
  • M-f 前进一个单词
  • M-b 后退一个单词
  • C-a 行首
  • C-e 行尾
  • C-v 下翻一页
  • M-v 上翻一页
  • M-< 文件头
  • M-> 文件尾

Edit

  • C-Space 设置开始标记
  • C-@ 设置开始标记(C-space可能被操作系统拦截)
  • M-w 复制标记区内容
  • C-y 帖粘
  • M-u 使光标处的单词大写
  • M-l 使光标处的单词小写
  • M-c 使光标处单词首字母大写
  • C-k 删除一行

Search

  • C-s 向下搜索
  • C-r 向上搜索
